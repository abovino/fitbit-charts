# Fitbit-charts

## Getting started

```bash
npm install
```

## ENV variables

Create a `.env` file in the root directory

```dosini
PORT=<PREFERRED_PORT>
FITBIT_CLIENT_ID=<YOUR_FITBIT_CLIENT_ID>
FITBIT_CLIENT_SECRET=<YOUR_FITBIT_CLIENT_SECRET>
FITBIT_CALLBACK_URL=<CALLBACK_URL_SPECIFIED_IN_FITBIT_API_SETTINGS>
```
